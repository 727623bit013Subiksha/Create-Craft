# 🎨 Create & Craft
A UI/UX mobile application designed to help users discover, save, and share creative craft ideas through a simple and user-friendly interface.

---

## 📖 Project Overview

Create & Craft is a mobile application that connects craft enthusiasts by providing an easy way to explore creative ideas, share projects, and find inspiration. The design focuses on a clean interface, smooth navigation, and an enjoyable user experience.

---

## 🎯 Problem Statement

Many users find it difficult to discover organized and inspiring craft ideas from different sources. The goal of this project was to create a single platform where users can easily browse, save, and share creative craft projects.

---

## 💡 Solution

- Designed a clean and modern mobile interface.
- Simplified navigation for a better user experience.
- Created interactive prototypes using Figma.
- Focused on intuitive layouts and visual consistency.

---

## ✨ Key Features

- Discover creative craft ideas
- Search and explore projects
- Save favorite crafts
- Share creative work
- User-friendly navigation
- Responsive mobile interface
